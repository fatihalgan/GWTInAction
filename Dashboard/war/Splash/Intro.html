Welcome to the GWT In Action Dashboard
This is the example that is built up through the GWT in Action book, By R Hanson and A Tacy and Published by Manning.

Please feel free to play around with the menus, drag and drop capabilities, the various component applications (under the Create menu), internationalization (under Help menu), etc.

Each component application shows off different aspects of GWT, you can find out what those are from the description menu item in the applications menu bar.

There are no real instructions; just click around and play - any comments are welcome on the book's web page (http://www.manning.com/hanson - also available under the bookmark menu).
